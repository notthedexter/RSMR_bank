# RSMR Bank
A modern Banking System Designed by Spring

Dependencies
-   Spring
    -   Web
    -   Security
    -   Lombok
    -   Validation
    -   Thymeleaf
    -   Mysql Connector
    -   Spring Data JPA
    -   Devtools
-   MariaDB

# MariaDB credintials
The mariadb credintials are stored in mariadb.sql
Just login to mysql as root and run
```
source mariadb.sql
```s
